サイト編集マニュアル

サイト編集 画像管理 ブログ投稿 お知らせ投稿

ログイン情報

ログインパスワード： fleurir-admin

1. サイト編集（edit.php）

各ページのテキストを編集できます。

タブ編集できる内容
料金メニュー全メニューの価格
営業情報営業時間、定休日、電話番号
TOPページWelcomeセクションの紹介文
サロン紹介オーナープロフィール（名前、肩書き、メッセージなど）
サービス各サービスの説明文（コスメリフト、フラットラッシュ、フェイシャルWAX）
商品商品の編集・追加・削除
各タブを選択 → テキストを入力 → 「変更を保存」をクリック

商品の管理

  • 編集：各商品の名前、説明、価格を変更できます
  • 追加：新しい商品を追加できます（画像もアップロード可能）
  • 削除：不要な商品を削除できます

2. 画像管理（images.php）

サイト内の画像を差し替えできます。

カテゴリ変更できる画像
TOPスライダーメインビジュアル 3枚
TOP料金メニューまつ毛エクステ、コスメリフト、眉WAX、フェイシャル
料金ページ各メニューの画像
サービスページコスメリフト、フラットラッシュ、フェイシャルWAX
商品ページ各商品の画像
変更する場所を選択 → 画像ファイルを選択 → 「アップロード」をクリック

3. ブログ投稿（blog.php）

新規投稿

  1. タイトルを入力
  2. カテゴリーを選択
  3. 画像を選択（任意）
  4. 抜粋を入力（一覧に表示される短い文章）
  5. 本文を入力
  6. 「投稿する」をクリック
画像は一覧のサムネイルと記事の両方に表示されます。
投稿すると自動的にブログ一覧に追加されます。

記事の削除

  1. ページ下部の「ブログ記事一覧・削除」で削除したい記事を探す
  2. 「削除」ボタンをクリック
  3. 確認ダイアログで「OK」をクリック
削除すると記事ファイルと関連画像が削除され、一覧からも自動的に削除されます。
※削除は取り消せません

4. お知らせ投稿（news.php）

  1. タイトルを入力
  2. 抜粋を入力（一覧に表示される短い文章）
  3. 本文を入力
  4. 「投稿する」をクリック
投稿すると自動的にお知らせ一覧に追加されます。
営業日変更やキャンペーン告知などにご利用ください。
